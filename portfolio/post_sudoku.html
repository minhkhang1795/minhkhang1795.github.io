Robotic Arm Solves Sudoku

September - December 2017

This was one of my projects in the Olin's Robotics Lab in the first semester of my freshman year. I joined the IRL team in the lab and the goal of the team is to control a robotic arm to interact with humans or to complete a task. I then teamed up with two other Oliners to work on the Sudoku project, which was quite challenging but very interesting to me because I love solving puzzles.

I was responsible for creating an algorithm to solve a Sudoku puzzle, finding a way to capture a Sudoku board including recognizing and locating the digits on the board, and actuating the robotic arm.

Check out the project on our Github repo.

Algorithm to solve a Sudoku puzzle

An algorithm to solve a sudoku puzzle was inspired by the paper: A Pencil-and-Paper Algorithm for Solving Sudoku Puzzles.

You can find the algorithm written in Python here: Sudoku algorithm.

A 4x4 sudoku board used in this project (24" x 24")

Capturing the board and processing the digits

The picture taken by the robotic arm would be processed using adaptive thresholding techniques in OpenCV.

Picture taken from the robotic arm

We then find the contour of the Sudoku board from the processed image and apply perspective transformation on the contour to get a perfect-square image of the Sudoku grid.

Processed image after adaptive thresholding
and perspective transformation

Finally, we crop out each digit and apply k-NN algorithm in OpenCV to predict the digits.

Final result

Using the Sudoku algorithm above, we'll find a solution for this puzzle.

A solved Sudoku puzzle

Robotic arm's actuation

In this project, the arm will firstly move to the center position above the Sudoku board to take a good picture of the board. After processing the picture and solving the Sudoku puzzle, the arm would then move to each empty cell and write the solution.
