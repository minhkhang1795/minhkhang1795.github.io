Robotic Arm Plays the Card Game Set

September - December 2017

This was another project in the Olin's Robotics Lab where I worked with three other members to make the robotic arm play the card game Set. I was mainly responsible for creating an algorithm to find a set and actuating the arm.

Check out the project on our Github repo.

Robotic Arm playing Set in action

Algorithm to find a set

The basic idea is to look through all possible combinations of 3 cards and check whether they make a set. The algorithm is encapsulated into a class, which can be found here.

Robotic arm's actuation

In this project, the arm will firstly move to the center position above the Set board to take a good picture of the board. After processing the picture and finding 3 cards that make a set, the arm would then move to those cards and pick them up.
